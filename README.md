# 簡易天氣預報App

<h3>功能介紹</h3>
● 選擇某個城市後，會顯示該城市當天的天氣情況，以及該城市未來七天的天氣情況。

● 點擊該城市當天的空氣情況後，會挑轉至另一頁顯示生活小提示。

<h3>天氣數據來源</h3>
免費七天天氣Api：<a href="https://yiketianqi.com/index/doc?version=v1">https://yiketianqi.com/index/doc?version=v1</a>
